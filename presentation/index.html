What We Built

Oracle Racing (Multiplayer)

An online, real-time racing game where players compete via a public MQTT broker. It features a persistent Hall of Fame using MQTT's `retain` feature, eliminating the need for a complex backend database.

  • Real-time WebSocket sync
  • Serverless Leaderboard
  • Pure Vanilla JS & Tailwind

FloodBoy & DustBoy Dashboards

Interactive dashboards tracking environmental sensors (Water Depth, Battery, PM2.5) by connecting directly to JIBCHAIN smart contracts using Viem ES modules.

  • Directly reads Blockchain state
  • Chart.js Data Visualization
  • Beautiful 2D & 3D Maps integration
