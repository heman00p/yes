## Lessons Learned
- **Architecture Constraints Shape Behavior**: ข้อจำกัดของ OS และ Security ก่อให้เกิดโครงสร้างการแก้ปัญหาที่สร้างสรรค์ (เช่น การทำ .bat หรือการพึ่งพามนุษย์ช่วยกดข้าม)
- **Camera Perspective (3D)**: การทำแอนิเมชันให้เนียน ไม่ใช่แค่ขยับตัวโมเดล แต่คือการเข้าใจว่ามุมมองของผู้มอง (Camera) ควรเป็นอย่างไรถึงจะถ่ายทอดเรื่องราวได้ครบถ้วน
- **Visuals Dictate Understanding**: ไม่ว่าข้อมูล GPS หรือ PM2.5 จะซับซ้อนแค่ไหน การแปลงมันมาเป็น Data Visualization เช่น Leaflet หรือ Chart.js ช่วยย่นระยะเวลาทำความเข้าใจของมนุษย์ได้มหาศาล
